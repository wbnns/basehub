## Overview

The `alloy/consensus` sub-crate (`base-alloy-consensus`) provides Base-specific consensus types, including custom transaction and receipt types with Base-specific fields. These types extend the standard Alloy consensus types for the OP Stack L2 environment.

## Key Dependencies

- `alloy-consensus` — Upstream consensus types
- `op-alloy-consensus` — OP Stack consensus types
