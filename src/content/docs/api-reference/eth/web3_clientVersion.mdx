Returns the version string of the node client software.

<Aside type="note">
This method is not exposed on every public endpoint. The Sepolia public node (`sepolia.base.org`) returns `-32601 Method not found`, while it works on mainnet.
</Aside>

## Parameters

None.

## Returns

| Field | Type | Description |
| --- | --- | --- |
| `result` | string | The client version string, formatted as `<client>/<version>-<commit>/<platform>/base/<base-version>`. Treat it as opaque — the format shifts with node software updates, so avoid parsing it. |

## Example

### Request

```json
{
  "jsonrpc": "2.0",
  "method": "web3_clientVersion",
  "params": [],
  "id": 1
}
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "reth/v1.11.3-2ac58a2/x86_64-unknown-linux-gnu/base/v0.9.0"
}
```
